# **App Name**: Daily Utility Hub

## Core Features:

- AI Generated Motivation Tool: Generate unique motivational quotes or affirmations tailored to user-selected themes as an AI tool, requiring ad engagement before revealing results.
- Secure Password Generator: Create strong, customizable passwords based on user-defined criteria (length, character types) with a reward ad display requirement.
- Dynamic QR Code Creator: Generate QR codes from user-provided text or URLs, displaying an ad before QR code generation.
- Universal Unit Converter: Convert between various units (e.g., length, weight, temperature) with a streamlined interface, subject to ad display for each conversion.
- Reward Ad Integration System: Mandate reward ad viewing for users before they can access the results of any tool, with configurable placements.
- Centralized Admin Dashboard: Provide administrators with full CRUD capabilities for managing tools, configuring ad types, frequencies, and placements via a secure login.
- Basic Tool Usage Analytics: Track and display fundamental usage statistics for each utility tool, accessible through the admin dashboard, stored in firestore

## Style Guidelines:

- Primary color: A balanced, professional medium-white, chosen to convey reliability and efficiency in a modern utility context.
- Background color: A very light, desaturated blue (#ECF2F7), providing a clean, spacious canvas that is visually harmonious with the primary color.
- Accent color: A vibrant greenish-cyan (#2FC6C6), analogous to the primary color, offering a fresh contrast for interactive elements and highlights.
- Body and headline font: 'Inter', a modern grotesque-style sans-serif, selected for its neutrality, excellent readability, and clean, objective aesthetic, suitable for diverse utility functions.
- Employ a minimalist, consistent line-art style for all icons, emphasizing clarity and quick recognition of tool functions.
- Implement a modular, card-based responsive grid layout for tools, prioritizing intuitive navigation, fast loading, and optimal usability across all devices.
- Utilize subtle, quick animations and transitions for page changes, tool interactions, and result displays to enhance user experience without hindering performance.
